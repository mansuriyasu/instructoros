# InstructorOS

Separate copy of the SparkOn Instructor Pro app for `https://instructoros.ca`.

## New Project Setup

This repository is intentionally not connected to the old SparkOn GitHub repo or live app.
Create a separate Firebase project, a separate hosting app/domain, and fresh environment variables before deploying.

Useful commands:

```bash
npm run dev
npm run build
```

More project notes for Codex are in `AGENTS.md`.
