# **App Name**: SparkOn Instructor Pro

## Core Features:

- Student Grid: Display student cards with status colors and search/filter options.
- Student Details Dialog: Show student information, comments, and payment history in a dialog. Includes edit, delete, and status change options.
- Student Management: Enable adding, editing, deleting, and importing/exporting student data as JSON files.
- Services Management: Manage services with name, price, and discount. Implement drag-and-drop reordering.
- Point of Sale (POS): Create payments with service selection, student assignment, price editing, and payment method options.
- Payment History: Display a history of payments with revenue reports and status toggling.
- Google Calendar Embed: Embed a Google Calendar via iframe with code input and save functionality.

## Style Guidelines:

- Primary color: Blue (#29ABE2), conveying professionalism and trust.
- Background color: Light blue (#E1F5FE), providing a clean and calming backdrop.
- Accent color: Teal (#008080), for interactive elements and highlights, complementing the primary blue.
- Body and headline font: 'Inter', a sans-serif font for modern and readable text.
- Use flat, line-based icons from a set like 'Feather' or 'Lucide' to ensure consistency and a modern look.
- Utilize a responsive grid layout for the student list and service management pages, ensuring adaptability across devices.
- Implement subtle transition animations for dialogs and page transitions to provide smooth navigation.
